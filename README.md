# Global Fiscal Monitor

## Project Overview
This project is an analytics solution designed to track ans analyze global government spending. It solves the "Double-Counting" problem in fiscal reporting by implementing a custom **RowType hierarchy** and automates financial calculations using a Python-based ETL engine.

### Key Technical Features
**Data Orchestration:** SQL Server (SSMS) utilized for relational modeling and data cleaning.
**Financial Engine:** Python script calculates **Year-on-Year (YoY) Growth %** and handles data grouping logic.
**Dashboard:** Interactive Power BI UI featuring a Decomposition Tree, Geospatial Analysis and automated KPI tracking.
**Zero-Error Logic:** Implemented a `RowType` toggle to separate 'Total' fiscal figures from 'Departmental' breakdowns.

## Tech Stack
**Database:** SQL Server
**Languages:** SQL, Python (Pandas, SQLAlchemy)
**Visualization:** Power BI Desktop
**Math:** Time-series analysis for fiscal growth forecasting.
